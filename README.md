# Quiet Luxury Portugal

Landing page premium para experiencias privadas de luxo em Portugal. Site estatico, sem frameworks, focado em performance e elegancia visual para um publico internacional 50+.

## Sobre o projecto

Quiet Luxury Portugal oferece experiencias de 5 dias totalmente hospedadas em Portugal: alojamento premium, gastronomia curada, natureza, cultura e transporte privado. O site serve como ponto de contacto inicial para potenciais clientes.

## Estrutura

```
assets/
  css/style.css       Estilos completos com responsive (3 breakpoints)
  js/main.js          Interactividade, animacoes de scroll, parallax, menu mobile
  img/                Logo SVG e PNG
index.html            Pagina principal com SEO (Open Graph, Twitter Card, Schema.org)
```

## Tecnologias

- HTML5, CSS3, JavaScript vanilla
- Google Fonts (Playfair Display + Inter)
- IntersectionObserver para animacoes de scroll
- CSS custom properties para temas
- Ken Burns no hero, parallax nas seccoes de destaque
- Formulario de contacto via WhatsApp

## Como usar

Abrir `index.html` no browser ou fazer deploy em qualquer servico de hosting estatico (Vercel, Netlify, GitHub Pages).
